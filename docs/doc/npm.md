# 官方插件总览

## 分类说明

* 核心：FaasJS 的核心，目前为 `@faasjs/func`
* 工具：辅助开发的工具，如测试、部署等
* 插件：为云函数提供额外功能的插件
* 服务商适配器：用于适配云服务商的适配器
