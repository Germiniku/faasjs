ESLint configuration for node.

## Install

    npm install @faasjs/eslint-config-recommended

Then put the following code into your `package.json` file:

```json
"eslintConfig": {
  "extends": [
    "@faasjs/recommend"
  ]
}
```

## Usage

    npm exec eslint .
