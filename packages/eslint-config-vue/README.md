ESLint configuration for Vue.

## Install

    npm install @faasjs/eslint-config-vue

Then put the following code into your `package.json` file:

```json
"eslintConfig": {
  "extends": [
    "@faasjs/vue"
  ]
}
```

## Usage

    npm exec eslint .
